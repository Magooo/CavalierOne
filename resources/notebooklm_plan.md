# NotebookLM Knowledge Base Plan

## Overview
We will leverage your **Cavalier One <-> NotebookLM** global connection to create intelligent, queryable knowledge bases for each core business function. This allows the AI agent to access approved, specific knowledge when helping you.

## 1. Structure: One Notebook per "Subject"
Instead of one giant notebook, create focused Notebooks for each domain to ensure high-quality answers.

### 📘 Notebook 1: "Cavalier Sales"
**Target Audience:** Sales Consultants, Marketing AI
**Content to Upload:**
*   `Sales-Training.pdf` (The Fast Keys Guide)
*   `Home_Designs_Master_List.pdf` (or brochures)
*   `Price_List_Shepparton.pdf`
*   `Objection_Handling_Scripts.docx`

**Use Case:**
*   "Draft an email to a client explaining why the Fast Keys price is fixed."
*   "What is the upgrade cost for the Malvern kitchen?"

### 📘 Notebook 2: "Cavalier Technical"
**Target Audience:** Construction, Estimating
**Content to Upload:**
*   `Inclusions_Specification_Sheet.pdf`
*   `Engineering_Standards.pdf`
*   `NCC_2025_Updates.pdf`

**Use Case:**
*   "Check if this inclusion meets the 2025 energy rating."

### 📘 Notebook 3: "Cavalier Admin & Process"
**Target Audience:** Office Staff, Management
**Content to Upload:**
*   `Admin_Workflow.pdf`
*   `Site_Start_Checklist.xlsx`
*   `Contract_Signing_Process.docx`

## 2. Execution Strategy

### Step 1: Create Notebooks (User Action)
1.  Go to [NotebookLM](https://notebooklm.google.com/).
2.  Click **New Notebook** -> Name it "Cavalier Sales".
3.  Click **Add Source** -> Upload the files listed above.
4.  Repeat for "Technical" and "Admin".

### Step 2: Connect Agent (Done)
*   The global connection is already established.
*   The agent can now use tools to "Query Notebook".

### Step 3: Workflow Integration
When asking the agent to perform a task, specify the context:
> "Using the **Cavalier Sales** knowledge base, write a response to this customer..."

## 3. Maintenance
*   **Monthly Review:** Update Price Lists and Specifications in the respective Notebooks.
*   **Archive Old Data:** Remove outdated PDFs to prevent "hallucination" of old prices.
